the folder includes php necessary for database connection.

(codes are taken from UCL Server Programming Course)
1. db_connection: connects to the database
2. db_functions: formating a response from the database
3. 1-mapdata: getting info on infected cities for the first map and converting to json
4. 2-mapdata: getting info on infected cities for the second map and converting to json
5. 3-mapdata: getting info on infected cities for the third map and converting to json
6. cities-mapdata: getting info on big cities of the region for the three maps and converting to json
7. h-mapdata: getting info on harbours and converting to json
8.t-mapdata: getting info on commercial networks and converting to json
